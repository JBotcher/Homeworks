ა) 17 → 10001
ბ) 34 → 100010
გ) 255 → 11111111


2. გადაიყვანეთ შემდეგი ორობითი რიცხვები ათობითში:

ა) 10101 → 21
ბ) 110011 → 51
გ) 10000000 → 128


3. განმარტეთ რას ნიშნავს ცვლადი C#-ში, როგორ შეგვიძლია მისი გამოცხადება და ინიციალიზაცია. მოიყვანეთ რამდენიმე მაგალითი:

ცვლადი არის მეხსიერებაში დაჯავშნილი ადგილი,რომელიც შეიძლება შეიცავდეს მონაცემებს, როგორიცაა რიცხვები, სიმბოლოები და ტექსტი.
C#-ში ცვლადების გამოცხადება ხდება ტიპის მითითებით, რაც მიანიშნებს, თუ რა ტიპის მონაცემებს მოითხოვს ცვლადი.

გამოცხადების მაგალითები:

int age;           // ცვლადის გამოცხადება (integer ტიპის)
string name;       // ცვლადის გამოცხადება (string ტიპის)
bool isStudent;    // ცვლადის გამოცხადება (boolean ტიპის)
double height;     // ცვლადის გამოცხადება (float ტიპის)

ინიციალიზაცია არის ცვლადისთვის საწყისი მნიშვნელობის მინიჭება.
მაგალითი:

int age = 25;           // მთელი რიცხვი
string name = "Alice";  // ტექსტი
bool isStudent = true;  // ბულეანული მნიშვნელობა
double height = 5.9;    // წილადი რიცხვი
